## Requirements
- Node.js v18
- Nx version 20

## Installation Steps
1. Install Node.js v18 from [Node.js official website](https://nodejs.org/).
2. Install Nx globally using the following command:
   ```bash
   npm install -g nx
   ```
3. Install project dependencies:
   ```bash
   npm install
   ```
4. Run the project:
   ```bash
   npx nx serve my-todos
   ```
